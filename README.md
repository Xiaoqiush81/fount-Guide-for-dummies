## ✨ 为什么选择 fount？

- **绝对自由**：支持原生 JS 动画/无过滤 HTML 渲染。
- **高度兼容**：支持 Risu (charX)、SillyTavern 等主流格式。
- **完全本地**：保护隐私，所有对话均在本地进行。
- **开源免费**：社区驱动，持续进化。
