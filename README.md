# fount 白痴手册

## 快速导航

- [**如何搭建**](./setup.md) — 从零开始安装 fount、导入角色、配置 AI 源
- [**如何创建角色**](./writing-char.md) — 编写原生 fount 角色卡（需 JavaScript 基础）
