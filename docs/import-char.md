# 导入角色

本章节详细介绍如何将角色导入到 fount 中。

## 1. 导入入口

1. 进入 fount 主界面。
2. 点击右上角的 **菜单按钮 (☰)**。
3. 在弹出的菜单中选择 **“组件”** -> **“导入”**。

## 2. 两种导入方式

### 方式 A：文件导入 (本地文件)

适用于你已经下载了角色包的情况。

- **支持格式**：
  - **fount 角色包** (`.zip`)：包含 `fount.json` 和 `main.mjs` 以及其他东西的完整包。
  - **酒馆角色卡** (`.png`, `.json`)：常见的 SillyTavern / Risu 格式角色卡。
- **操作**：
  1. 点击左侧的上传区域。
  2. 选择你的文件。
  3. 等待导入完成。

### 方式 B：链接导入 (网络链接)

适用于从 GitHub 或角色分享站直接导入。

- **支持来源**：GitHub 仓库链接、RisuRealm 链接、Chub 链接等。
- **操作**：
  1. 在右侧的文本框中粘贴链接。
  2. **支持批量**：每一行粘贴一个链接。
  3. 点击 **“导入”** 按钮。

> **❓ 常见问题**
>
> - **导入后没显示？** 别慌，尝试刷新一下网页 (F5) 就行。
> - **显示炸了/报错？** 可能是网络问题导致资源下载不全，或者角色卡本身格式有误。尝试删除后重新导入试试。
